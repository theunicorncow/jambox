enum gnsdk_musicid_query_status_t

Indicates a MusicID callback function status value.

Enumerator:
gnsdk_musicid_status_unknown 

MusicID query status is unknown.

gnsdk_musicid_status_query_begin 

Begin MusicID query.

gnsdk_musicid_status_connecting 

MusicID query is starting a network connection

gnsdk_musicid_status_sending 

MusicID is sending query data.

gnsdk_musicid_status_receiving 

MusicID is receiving query data.

gnsdk_musicid_status_disconnected 

MusicID query is disconnecting a network connection

gnsdk_musicid_status_query_complete 

MusicID query handle is completing the query process.

gnsdk_musicid_status_query_delete 

MusicID query handle is about to be deleted.

GNSDK_DECLARE_HANDLE ( gnsdk_musicid_query_handle_t  )

Handle to a single MusicID query that is created by the gnsdk_musicid_query_create(). The application must create this handle for each MusicID query it needs to execute.
