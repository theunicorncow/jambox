gnsdk_error_t GNSDK_API gnsdk_playlist_attributes_count ( gnsdk_uint32_t *  p_count)

Retrieves the total number of registered attributes for use in the Playlist system.

Parameters
p_count[out] Pointer to receive total number of registered Playlist attributes
Returns
PLERR_NotInited gnsdk_playlist_initialize was not successfully initiated
PLERR_InvalidArg p_count is equal to NULL
PLERR_NoError System-specific registered attributes were successfully counted

Remarks:

Playlist comes with several supported attributes. Collection summaries may have been created with a different set or sub set of these attributes. Only attributes supported by both the system and a collection summary can be used by a Playlist definition.
