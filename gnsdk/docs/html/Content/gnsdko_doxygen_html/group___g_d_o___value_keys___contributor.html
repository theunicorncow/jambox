#define GNSDK_GDO_VALUE_BIOGRAPHY_LANGUAGE   "gnsdk_val_biography_language"

Retrieves the available language value for a returned GNSDK_GDO_VALUE_BIOGRAPHY object.

Remarks:

Use this value to retrieve the actual language value of a returned object. The language value of a returned object depends on availability. Information in the language set for the locale may not be available, and the object's information may be available only in its default official language. For example, if a locale's set language is Spanish, but the object's information is available only in English, this value returns as English.
