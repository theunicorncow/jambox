#define GNSDK_PLAYLIST_MORELIKETHIS_OPTION_RANDOM   "gnsdk_playlist_morelikethis_option_random"

Option for querying/specifying the random value used in calculating a "More Like This" playlist. The value for this key must evaluate to a number greater than 0. The recommended range is 1 - 100, but you can choose any number. This number will be the seed for the randomization. You can re-create a playlist by choosing the same number for the randomization seed. Choosing different numbers will create different playlists.
