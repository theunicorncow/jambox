Detailed Description

The GNSDK Link library provides access to enhanced and third-party music and video data that is linked to Gracenote data. Examples of this additional data include cover artwork, artist biographies, song lyrics, and more.

This library can also return any custom information that a customer has linked to Gracenote data, such as their own media identifiers.

This library is meant for applications that must receive these enhanced content services. The application provides Gracenote identifiers (derived from previous Gracenote identification results) to the Link library to retrieve the additional content.

Note: Customers must be entitled to implement use of the enhanced Link data in an application. Contact your Gracenote Professional Services representative for questions about this entitlement.

Image Availability

At the time of this guide's publication, only limited content is available for Album Artist image queries. Contact your Gracenote Professional Services representative for updates on the latest supported images.

The APIs described in this section are defined in gnsdk_link.h.
