gnsdk_cstr_t GNSDK_API gnsdk_manager_get_build_date ( void  )

Retrieves the GNSDK Manager's build date string.

Returns
Note Build date string of the format: YYYY-MM-DD hh:mm UTC This API can be called at any time, even before initialization and after shutdown. The returned string is a constant. Do not attempt to modify or delete. Example: 2008-02-12 00:41 UTC
gnsdk_cstr_t GNSDK_API gnsdk_manager_get_globalid_magic ( void  )

Retrieves the GNSDK Manager Global-ID Magic number string. This API can be called at any time, even before initialization and after shutdown. The returned string is a constant. Do not attempt to modify or delete. Example: GNN1
