gnsdk_error_t GNSDK_API gnsdk_manager_string_free ( gnsdk_str_t  string)

Frees the memory associated with an GNSDK Manager-provided serialized string or XML string. Passing a null string value returns SDKMGRERR_NoError.

Parameters
string[in] Pointer to string buffer to free
Returns
SDKMGRERR_NotInited gnsdk_manager_initialize() was not successfully initiated
SDKMGRERR_HandleObjectInvalid Given string parameter is not a string value returned by GNSDK that should be freed
SDKMGRERR_NoError String was successfully freed
