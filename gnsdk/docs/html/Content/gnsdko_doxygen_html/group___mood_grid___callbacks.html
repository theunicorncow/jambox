typedef gnsdk_void_t(GNSDK_CALLBACK_API * gnsdk_moodgrid_callback_fn)(gnsdk_void_t *user_data, gnsdk_moodgrid_presentation_handle_t moodgrid_presentation_handle, gnsdk_moodgrid_status_t status, gnsdk_size_t bytes_done, gnsdk_size_t bytes_total, gnsdk_bool_t *p_abort)

Mood Grid callback function for status updates as content is retrieved.

Parameters
user_data[in] Pointer to data passed in to gnsdk_moodgrid_presentation_create function through the callback_userdata parameter. You must cast this pointer from the gnsdk_void_t type to its original type to be accessed properly.
moodgrid_presentation_handle[in] The presentation handle that the callback operates on.
status[in] One of gnsdk_moodgrid_status_t values.
bytes_done[in] Current number of bytes transferred. Set to a value greater than 0 to indicate progress, or 0 to indicate no progress.
bytes_total[in] Total number of bytes to be transferred. Set to a value greater than 0 to indicate progress, or 0 to indicate no progress.
p_abort[out] Set dereferenced value to GNSDK_TRUE to abort the operation that calls the callback.
