GNSDK C API Reference Documentation

This reference describes the C APIs provided by the GNSDK libraries. The reference is organized into the following sections:

  • GNSDK Manager: Provides core functionality necessary to all Gracenote libraries.
  • Gracenote Data Object (GDO): Provides access to Gracenote Data Objects (GDOs), which are used to pass data between libraries of the SDK.
  • Link: Provides access to enhanced or third-party data that is linked to Gracenote data.
  • Lists and Locales: Provides access to list structures and locale identifiers.
  • Mood Grid: Provides spatially-aware music organization features to manage a user's music collection based on a set of Gracenote attributes.
  • Music: Provides CD TOC, text, file-based, and fingerprint recognition for music.
  • Playlist: Provides functionality to generate playlists when integrated with the GNSDK MusicID or MusicID-File library (or both).
  • Setup: Provides functionality for initializing GNSDK and setting up a local database.
  • Storage and Caching: Provides caching functionality for queries and content.

The APIs described in this Reference are a superset of the GNSDK C APIs. Not all products support the entire data model. Contact your Gracenote Professional Services representative for more information.
