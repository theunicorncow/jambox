gnsdk_cstr_t GNSDK_API gnsdk_playlist_get_build_date ( void  )

Retrieves the Playlist library's build date string.

Returns
Note Build date string of the format: YYYY-MM-DD hh:mm UTC

Remarks:

This API can be called at any time, even before initialization and after shutdown. The returned string is a constant. Do not attempt to modify or delete. Example Example build date string: 2008-02-12 00:41 UTC

gnsdk_cstr_t GNSDK_API gnsdk_playlist_get_version ( void  )

Retrieves the Playlist library's version string. This API can be called at any time, even before initialization and after shutdown. The returned string is a constant. Do not attempt to modify or delete. Example: 1.2.3.123 (Major.Minor.Improvement.Build) Major: New functionality Minor: New or changed features Improvement: Improvements and fixes Build: Internal build number

gnsdk_error_t GNSDK_API gnsdk_playlist_initialize ( gnsdk_manager_handle_t  sdkmgr_handle)

Initializes the GNSDK Playlist library.

Parameters
sdkmgr_handle[in] Handle from a successful gnsdk_manager_initialize() call
Returns
PLERR_InvalidArg Given sdkmgr_handle parameter is null
PLERR_HandleObjectInvalid Given sdkmgr_handle is not a valid GNSDK Manager handle
PLERR_InitFailed Initialization failed
PLERR_NoError Initialization succeeded

Remarks:

This function must be successfully called before any calls to other Playlist APIs will succeed. This function can be called multiple times, however, all successful calls must be paired with a call to gnsdk_playlist_shutdown(). If a call to gnsdk_playlist_initialize returns an error, then its corresponding gnsdk_playlist_shutdown call must not be called.

gnsdk_error_t GNSDK_API gnsdk_playlist_shutdown ( void  )

Shuts down and releases resources for the Playlist library.

Returns
PLERR_NotInited gnsdk_playlist_initialize was not successfully initiated
PLERR_NoError Shut down succeeded

Remarks:

All gnsdk_playlist_shutdown calls must be paired with an initial call to gnsdk_playlist_initialize(). The final gnsdk_playlist_shutdown call results in the library being completely shut down; all Playlist APIs cease to function until the library is again initialized.
