gnsdk_cstr_t GNSDK_API gnsdk_musicid_get_build_date ( void  )

Retrieves the MusicID SDK's build date string.

Returns
Note Build date string of the format: YYYY-MM-DD hh:mm UTC

Remarks:

This API can be called at any time, even before initialization and after shutdown. The returned string is a constant. Do not attempt to modify or delete. Example build date string: 2008-02-12 00:41 UTC

gnsdk_cstr_t GNSDK_API gnsdk_musicid_get_version ( void  )

Retrieves the MusicID SDK version string.

Returns
version string if successful
GNSDK_NULL if not successful

Remarks:

This API can be called at any time, even before initialization and after shutdown. The returned string is a constant. Do not attempt to modify or delete. Example version string: 1.2.3.123 (Major.Minor.Improvement.Build) Major: New functionality Minor: New or changed features Improvement: Improvements and fixes Build: Internal build number

gnsdk_error_t GNSDK_API gnsdk_musicid_initialize ( gnsdk_manager_handle_t  sdkmgr_handle)

Initializes the MusicID library.

Parameters
sdkmgr_handle[in] Handle from a successful gnsdk_manager_initialize() call
Returns
MIDERR_InvalidArg Given sdkmgr_handle parameter is invalid
MIDERR_HandleObjectInvalid Given sdkmgr_handle is not a valid GNSDK Manager handle
MIDERR_InitFailed Initialization failed
MIDERR_NoError Initialization succeeded

Remarks:

This function must be successfully called before any calls to other MusicID APIs will succeed. This function can be called multiple times, however, all successful calls must be paired with a call to gnsdk_musicid_shutdown(). If a call to gnsdk_musicid_initialize returns an error, then its corresponding gnsdk_musicid_shutdown() call must not be called.

gnsdk_error_t GNSDK_API gnsdk_musicid_shutdown ( void  )

Shuts down and release resources for the MusicID library.

Returns
MIDERR_NotInited gnsdk_musicid_initialize was not successfully initiated
MIDERR_NoError Shut down succeeded

Remarks:

All gnsdk_musicid_shutdown calls must be paired with an initial call to gnsdk_musicid_initialize(). The final gnsdk_musicid_shutdown call results in the library being completely shut down; all MusicID APIs cease to function until the library is again initialized.
