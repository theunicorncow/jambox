enum gnsdk_link_content_type_t

Indicates available content types that can be retrieved for Albums, Lyrics, Tracks, Video Products, Contributors, Works, Seasons, or Series (or some combination of these object types). Not all content types are available for all objects.

Enumerator:
gnsdk_link_content_unknown 

Indicates an invalid content type.

gnsdk_link_content_cover_art 

Retrieves cover artwork; this is Album-level and Video Product-level content.

Use with GNSDK_GDO_TYPE_ALBUM and GNSDK_GDO_TYPE_VIDEO_PRODUCT.

Do not use with GNSDK_GDO_TYPE_VIDEO_WORK, GNSDK_GDO_TYPE_VIDEO_SERIES, GNSDK_GDO_TYPE_VIDEO_SEASON, or GNSDK_GDO_TYPE_CONTRIBUTOR.

gnsdk_link_content_genre_art 

Retrieves artwork for the object's primary genre; this is Album-level and Track-level content.

gnsdk_link_content_review 

Retrieves a review for the object; this is Album-level content.

gnsdk_link_content_biography 

Retrieves a biography for the object; this is Album-level content.

gnsdk_link_content_artist_news 

Retrieves news for the artist; this is Album-level content.

gnsdk_link_content_lyric_xml 

Retrieves lyrics in XML form; this is Lyric-level and Track-level data.

gnsdk_link_content_lyric_text 

Retrieves lyrics in plain text form; this is Lyric-level and Track-level content.

gnsdk_link_content_dsp_data 

Retrieves DSP content; this is Track-level content.

gnsdk_link_content_comments_listener 

Retrieves listener comments; this is Album-level content.

gnsdk_link_content_comments_release 

Retrieves new release comments; this is Album-level content.

gnsdk_link_content_news 

Retrieves news for the object; this is Album-level content.

gnsdk_link_content_image 

Retrieves an image for specific Video types.

Use with GNSDK_GDO_TYPE_VIDEO_WORK, GNSDK_GDO_TYPE_VIDEO_SERIES, GNSDK_GDO_TYPE_VIDEO_SEASON, and GNSDK_GDO_TYPE_CONTRIBUTOR.

Do not use with GNSDK_GDO_TYPE_VIDEO_PRODUCT.

Note: At the time of this guide's publication, support for Video Explore Seasons and Series image retrieval through Seasons, Series, and Works queries is limited. Contact your Gracenote Professional Services representative for updates on the latest supported images.

gnsdk_link_content_image_artist 

Retrieves an artist image for the object; this is Album-level content.

Use with GNSDK_GDO_TYPE_ALBUM.

Note: At the time of this guide's publication, the available Album artist images are limited. Contact your Gracenote Professional Services representative for updates on the latest supported images.

enum gnsdk_link_data_type_t

Indicates possible data formats for the retrieved content.

Enumerator:
gnsdk_link_data_unknown 

Indicates an invalid data type.

gnsdk_link_data_text_plain 

Indicates the content buffer contains plain text data (null terminated).

gnsdk_link_data_text_xml 

Indicates the content buffer contains XML data (null terminated).

gnsdk_link_data_number 

Indicates the content buffer contains a numerical value (gnsdk_uint32_t). Unused.

gnsdk_link_data_image_jpeg 

Indicates the content buffer contains jpeg image data.

gnsdk_link_data_image_png 

Indicates the content buffer contains png image data.

gnsdk_link_data_binary 

Indicates the content buffer contains externally defined binary data.
