GNSDK_DECLARE_HANDLE ( gnsdk_user_handle_t  )

Handle to user identification. Created by the gnsdk_manager_user_create() APIs. An application must create this handle for each unique user that calls the SDKs. This handle must be released by gnsdk_manager_user_release().
