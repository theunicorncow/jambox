gnsdk_error_t GNSDK_API gnsdk_playlist_storage_load_collection ( gnsdk_cstr_t  collection_name,
gnsdk_playlist_collection_handle_t *  ph_collection 
)

Loads a Playlist Collection Summary from local storage into memory for use. Applications cannot access a collection summary directly from storage; it must first be loaded into memory. Use this function to load a collection summary from the application-specific storage solution or (if implemented) the GNSDK SQLite library implementation.

Parameters
collection_name[in] Collection summary to load into memory
ph_collection[out] Pointer to receive a collection summary handle
Returns
PLERR_NotInited gnsdk_playlist_initialize was not successfully initiated
PLERR_NotFound Given collection_name is unavailable
PLERR_InvalidArg Given collection_name is empty or NULL, or given ph_collection is NULL
PLERR_LibraryNotLoaded Storage module was not successfully initialized
PLERR_NoError Collection handle was successfully loaded

Note:

Be sure to initialize the application's storage module prior to using this function.

gnsdk_error_t GNSDK_API gnsdk_playlist_storage_remove_collection ( gnsdk_cstr_t  collection_name)

Permanently removes a Playlist collection summary from local storage. Use this function to remove a collection summary name from storage. This function only removes the named collection from storage. If the collection currently exists as a handle, the handle and referenced collection will not be affected, and must still be released when their use is complete. The application is free to store the same named collection again after removing it.

Parameters
collection_name[in] Collection summary to delete from storage
Returns
PLERR_NotInited gnsdk_playlist_initialize was not successfully initiated
PLERR_NotFound Given collection_name is invalid
PLERR_InvalidArg Given collection_name is empty
PLERR_LibraryNotLoaded Storage module was not successfully initialized
PLERR_NoError Collection summary was successfully removed from storage

Note:

Be sure to initialize the application's storage module prior to using this function.
