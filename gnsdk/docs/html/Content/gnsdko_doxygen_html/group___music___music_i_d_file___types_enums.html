GNSDK_DECLARE_HANDLE ( gnsdk_musicidfile_query_handle_t  )

Handle to query instance for MusicID-File. The application creates this handle for each set of files designated for MusicID-File recognition.

Remarks:

An application may create as many of these handles as needed using gnsdk_musicidfile_query_create()

GNSDK_DECLARE_HANDLE ( gnsdk_musicidfile_fileinfo_handle_t  )

Handle containing all input and output data for a single file. Each handle is associated with a gnsdk_musicidfile_query_handle_t.

Remarks:

An application may create this handle as needed. This handle is owned by the gnsdk_musicidfile_query_handle_t it was created with, and is automatically destroyed when gnsdk_musicidfile_query_handle_t is destroyed.
