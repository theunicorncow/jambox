const gnsdk_error_info_t* GNSDK_API gnsdk_manager_error_info ( void  )

Retrieves any available extended error information for the current thread. To retrieve any extended error information, the application must call this function immediately after receiving an error from a Gracenote SDK API. Though not all APIs provide extended error information, at a minimum, the structure contains the description string for the error.

Returns
Note Returns a pointer to the populated gnsdk_error_info_t structure.
