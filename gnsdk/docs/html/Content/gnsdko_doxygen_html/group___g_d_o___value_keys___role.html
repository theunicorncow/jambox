#define GNSDK_GDO_VALUE_ROLE   "gnsdk_val_role"

Retrieves a role value from a GDO.

Remarks:

This value shows the role that a person played in a music or video production; for example, singing, playing an instrument, acting, directing, and so on.

Note:

For music credits, the absence of a role for a person indicates that person is the primary artist, who may have performed multiple roles. This is a list-based value requiring that the list be loaded into memory through a successful call to gnsdk_manager_locale_load.

To render locale-dependent information for list-based values, the application must call gnsdk_manager_locale_load and possibly also gnsdk_sdkmanager_gdo_set_locale. The application returns a LocaleNotLoaded message when locale information is not set prior to a request for list-based value information.
