gnsdk_cstr_t GNSDK_API gnsdk_link_get_build_date ( void  )

Retrieves Link library build date string.

Returns
Note Build date string of the format: YYYY-MM-DD hh:mm UTC

Remarks:

This API can be called at any time, even before initialization and after shutdown. The returned string is a constant. Do not attempt to modify or delete.

Example build date string: 2008-02-12 00:41 UTC

gnsdk_cstr_t GNSDK_API gnsdk_link_get_version ( void  )

Retrieves the Link library version string. This API can be called at any time, even before initialization and after shutdown. The returned string is a constant. Do not attempt to modify or delete.

Example: 1.2.3.123 (Major.Minor.Improvement.Build)
Major: New functionality
Minor: New or changed features
Improvement: Improvements and fixes
Build: Internal build number

gnsdk_error_t GNSDK_API gnsdk_link_initialize ( gnsdk_manager_handle_t  sdkmgr_handle)

Initializes the Link library. This function must be successfully called before any calls to other Link APIs will succeed. This function can be called multiple times, however, all successful calls must be paired with a call to gnsdk_link_shutdown(). If a call to gnsdk_link_initialize returns an error, then its corresponding gnsdk_link_shutdown() call must not be called.

Parameters
sdkmgr_handle[in] Handle from successful gnsdk_manager_initialize call
Returns
LINKERR_InvalidArg - Given sdkmgr_handle parameter is null.
LINKERR_HandleObjectInvalid - Given sdkmgr_handle is not a valid GNSDK Manager handle
LINKERR_InitFailed - Initialization failed
LINKERR_NoError - Initialization succeeded
gnsdk_error_t GNSDK_API gnsdk_link_shutdown ( void  )

Shuts down the Link library. All gnsdk_link_shutdown calls must be paired with an initial call to gnsdk_link_initialize(). The final gnsdk_link_shutdown call results in the library being completely shut down; all Link APIs cease to function until the library is again initialized.

Returns
LINKERR_NotInited gnsdk_link_initialize() was not successfully initiated
LINKERR_NoError Shut down succeeded
