#define GNSDK_GDO_VALUE_ALBUM_DISC_IN_SET   "gnsdk_val_albumdiscset"

Retrieves a disc number value from a GDO.

Remarks:

For album types, this retrieves the volume number of this album in a multi-volume set; note that this value is not always available. As this is a unique value, the ordinal parameter for gnsdk_manager_gdo_value_get must be set to 1 to retrieve the first (or only) value instance.

#define GNSDK_GDO_VALUE_COLLABORATOR_RESULT   "gnsdk_val_collab_result"

Retrieves a value indicating if a GDO contributor response result matched on the primary collaborator for the input collaboration string. For example, if the input text was "John feat. Joe" and the returned contributor match is the John's contributor object instead of the "John feat. Joe" collaboration contributor object. Returns 1 if this is a collaborator match, 0 if not.

Remarks:

Available for the following music types:

  • GNSDK_GDO_TYPE_CONTRIBUTOR

#define GNSDK_GDO_VALUE_TRACK_MATCHED_POSITION   "gnsdk_val_track_matched_pos"

Retrieves a numeric value indicating the position (in milliseconds) in the Track where the GNFPX fingerprint match occurs.

Remarks:

The Track Position Match value indicates at what track millisecond the GNFPX fingerprint begins to match, and is returned by using gnsdk_manager_gdo_value_get.

Note:

This value is retrieved only when using the GNFPX fingerprint data type. These code examples show only two of several possible ways to retrieve a Track Position Match value from Album and Track GDOs. Function call sequence using an Album GDO:

  1. gnsdk_musicid_query_create
  2. gnsdk_musicid_query_fingerprint_begin
  3. gnsdk_musicid_query_fingerprint_write
  4. gnsdk_musicid_query_fingerprint_end
  5. gnsdk_musicid_query_find_albums
  6. gnsdk_sdkmanager_gdo_child_get(result_gdo, GNSDK_GDO_CHILD_ALBUM, 1, &album_gdo)
  7. gnsdk_sdkmanager_gdo_value_get(album_gdo,GNSDK_GDO_VALUE_TRACK_MATCHED_POSITION,1,...)

When retrieving a Track Position Match value from a Track, use multiple calls to gnsdk_sdkmanager_gdo_child_get to access the related Album GDO type from the Track GDO type. Function call sequence using a Track GDO:

  1. gnsdk_musicid_query_create
  2. gnsdk_musicid_query_fingerprint_begin
  3. gnsdk_musicid_query_fingerprint_write
  4. gnsdk_musicid_query_fingerprint_end
  5. gnsdk_musicid_query_find_tracks
  6. gnsdk_sdkmanager_gdo_child_get(result_gdo, GNSDK_GDO_CHILD_TRACK, 1, &track_gdo)
  7. gnsdk_sdkmanager_gdo_child_get(track_gdo, GNSDK_GDO_CHILD_ALBUM, 1, &album_gdo)
  8. gnsdk_sdkmanager_gdo_value_get(album_gdo,GNSDK_GDO_VALUE_TRACK_MATCHED_POSITION,1,...)

You must be licensed and entitled for MusicID-Stream. For questions on implementing this functionality or licensing, contact your Gracenote Professional Services representative.
