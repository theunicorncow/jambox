GNSDK_DECLARE_HANDLE ( gnsdk_manager_handle_t  )

Primary handle to GNSDK Manager. Retrieved from GNSDK Manager's gnsdk_manager_initialize() API. An application must pass this handle to each Gracenote SDK on initialization. This handle does not have to be released.
