typedef gnsdk_void_t(GNSDK_CALLBACK_API * gnsdk_manager_logging_callback_fn)(const gnsdk_void_t *callback_data, gnsdk_uint16_t package_id, gnsdk_uint32_t filter_mask, gnsdk_cstr_t format, va_list argptr)

Manually manage Gracenote SDK logging messages.

Parameters
callback_data[in] Pointer to data passed in to the gnsdk_manager_logging_enable_callback function through the callback_data parameter. This pointer must be cast from the gnsdk_void_t type to its original type to be accessed properly.
package_id[in] ID of package sending the log message
mask[in] Filter mask of the type of log message
format[in] Format string of the log message
argptr[in] Set of arguments for the format string

Remarks:

This callback can be supplied to gnsdk_manager_logging_enable_callback.
